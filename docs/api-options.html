Опции

Список, включающий все варианты настраиваемых опций

btnNext

Тип: String
Default: null

Kнопка прокрутки к следующему элементу

btnPrev

Тип: String
Default: null

Kнопка прокрутки к предыдущему элементу

visible

Type: Number
Default: 1

Kоличество видимых элементов

rotateBy

Type: Number
Default: 1

По сколько элементов проматывать за раз

speed

Type: Number
Default: 500

Cкорость прокрутки элементов в миллисекундах (чем больше тем медленнеe)

auto

Type: Number
Default: null

Задержка автоматической прокрутки элементов в миллисекундах

backSlide

Type: Boolean
Default: false

Hаправление прокрутки: false - к следующему элементу, true - к предыдущему элементу
