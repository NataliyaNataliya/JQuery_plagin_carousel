Установка

Подключение CSS

Сначала включите файл CSS в HTML head:

<link rel="stylesheet" href="style.css">

Подключение JS

Подключите скрипты в указанном порядке перед закрывающим тегом body

<script src="jquery-3.6.0.min.js"></script>
 <script src="carousel.js"></script>

Структура HTML

Вместо ul может быть любой блочный элемент, в который могут быть вложены любые блочные элементы 

<!-- Set up your HTML -->
<ul class="carousel"
    <li> 1 </li>
    <li> 2 </li>
    <li> 3 </li>
    <li> 4 </li>
    <li> 5 </li>
    <li> 6 </li>
    <li> 7 </li>
</ul>

Кнопки навигации могут располагаться где угодно на странице

Вызов плагина

Теперь напишите такой вызов:

<script>
        $(function () {
           $('.container').pluginCarousel();
         });
</script>
